What is HTML?

HTML stands for HyperText Markup Language. It is the standard language used to create web pages. HTML provides the structure for web documents, which includes elements like headings, paragraphs, links, images, etc.

HTML is not a programming language, it is a markup language used to structure content on the web.

HTML Structure

HTML documents are made up of elements, and these elements are wrapped in tags. A basic HTML document structure looks like this:


<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <h1>This is a Heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

Above is the basic structure of an HTML document. It contains:

  • <!DOCTYPE html> - Declares the document type
  • <html> - The root element of an HTML page
  • <head> - Contains meta-information about the page (like title)
  • <body> - Contains the content of the page (headings, paragraphs, etc.)

HTML Tags

HTML tags are the building blocks of web pages. Each tag has a specific function. Some of the most commonly used tags are:

  • <h1> to <h6> - Defines headings
  • <p> - Defines a paragraph
  • <a> - Defines a hyperlink
  • <img> - Embeds an image
  • <div> - Defines a division or section
  • <span> - Defines a small section of content

HTML Attributes

Attributes provide additional information about HTML elements. They are always specified in the opening tag and are written as name/value pairs.

For example, the <a> tag has an href attribute to specify the destination of the link:


<a href="https://www.example.com">Visit Example</a>

In this example, the href attribute specifies the URL the link should point to.

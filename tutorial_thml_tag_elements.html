What are HTML Tags and Elements?

HTML tags are the building blocks of web pages. Tags are used to define elements on the page, such as headings, paragraphs, links, images, and more.

An HTML element generally consists of an opening tag, content, and a closing tag. For example: <p>This is a paragraph</p>.

Common HTML Tags

<h1> - Heading Tag

This is an example of a heading tag. It is typically used for the main heading of a page.

<h1>This is a Heading</h1>

Output: This is a Heading

<p> - Paragraph Tag

The <p> tag is used for defining a paragraph.

<p>This is a paragraph.</p>

Output: This is a paragraph.

<a> - Anchor Tag (Link)

The <a> tag is used to create hyperlinks to other pages or websites.

<a href="https://www.example.com">Visit Example</a>

Output: Visit Example

HTML Elements

<img> - Image Element

The <img> tag is used to embed an image in a webpage.

<img src="https://via.placeholder.com/150" alt="Placeholder Image">

Output:

<ul> - Unordered List Element

The <ul> tag is used to define an unordered list.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
  <li>Item 3</li>
</ul>

Output:

  • Item 1
  • Item 2
  • Item 3
